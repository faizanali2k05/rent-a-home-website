About the Project

Tech Stack: Built with HTML5, CSS3, Bootstrap 5, JavaScript (ES Modules), SQL (PostgreSQL via Supabase), and Supabase Authentication & Storage. A lightweight rental platform demonstrating property listings, bookings, reviews, and comprehensive landlord/tenant management flows. Landlords can create, edit, and delete their property listings with full RLS-based security.
